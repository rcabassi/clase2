1. Títulos y Énfasis
Títulos: Se usan almohadillas o hashes (#). A más hashes, menor el tamaño.

# Título 1

## Título 2

Negrita: Rodea el texto con dos asteriscos: **texto**.

Cursiva: Rodea el texto con un asterisco: *texto*.

Tachado: Rodea el texto con dos virgulillas: ~~texto~~.

2. Listas
Desordenadas: Usa asteriscos, guiones o signos de suma.

* Elemento A

* Elemento B

Ordenadas: Simplemente usa números seguidos de un punto.

1. Primero

2. Segundo

3. Enlaces e Imágenes
Enlaces: [Texto del enlace](URL)

Imágenes: Es igual al enlace pero con un signo de exclamación al inicio: ![Texto alternativo](URL de la imagen)

4. Código y Citas
Código en línea: Rodea con una comilla simple invertida: `código`.

Bloques de código: Usa tres comillas invertidas (```) al inicio y al final.

Citas: Usa el signo mayor que: > Esto es una cita.

5. Otros elementos
Línea horizontal: Tres guiones seguidos ---.

Tablas: Se crean usando barras verticales | para separar columnas y guiones - para la fila de encabezado.
